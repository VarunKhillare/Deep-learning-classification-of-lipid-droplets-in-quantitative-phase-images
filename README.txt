Steps to run this project:

Step-1: Download the dataset from the below link: 
https://verso.uidaho.edu/esploro/outputs/dataset/Data-from-Deep-learning-classification-of/996762915601851?institution=01ALLIANCE_UID

Step-2: First run the code train_ffnn.py file but before that make changes in the path directory as to where you have saved the dataset 
	and where you want to save the output models.

Step-3: Next run the validate.py , change the path directories accordingly.

Step-4: Run the evaluate.py and metrics.py , adjust the path directory.

Note: Keep the utils.py in the same folder as the other files as many functions are called from there.
